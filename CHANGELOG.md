# Changelog

All notable changes to eDeDo will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of eDeDo
- Color-coded enemy system (Blue/Purple/Red for 1/2/3 HP)
- Directional collision detection (jump on head vs side collision)
- Portal system with enemy defeat counter (15 enemies to win)
- Energy-based shooting system (no ammo, uses energy)
- Passive energy regeneration (30%/s after 1s cooldown)
- Procedural level generation
- Full gamepad support
- Particle effects system
- Procedural audio generation
- Lives system with heart pickups
- Pause menu
- Game over screen

### Changed
- Enemies now display happy faces instead of HP numbers
- Enemy colors change dynamically based on current HP
- Missile projectiles annihilate enemy bullets on contact

### Fixed
- Controller sensitivity issues
- Energy regeneration timing
- Collision detection edge cases
